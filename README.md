# Task Tracker Web Application
A simple web application that helps users organize and manage their daily tasks.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Task counter (total tasks and completed tasks)
- Clean and simple user interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage

## How It Works

1. Enter a task in the input field
2. Click the Add button
3. The task will appear in the task list
4. Mark tasks as completed using the checkbox
5. Delete tasks using the delete button
